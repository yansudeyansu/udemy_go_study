# Lesson 6: 演算子

## 算術演算子
1. 基本的な算術演算子
   - `+` (加算)
   - `-` (減算) 
   - `*` (乗算)
   - `/` (除算)
   - `%` (剰余)

2. インクリメント・デクリメント
   - `++` (インクリメント)
   - `--` (デクリメント)

3. 代入演算子
   - `+=` (加算代入)
   - `-=` (減算代入)
   - `*=` (乗算代入)
   - `/=` (除算代入)
   - `%=` (剰余代入)

## 比較演算子
1. 数値の比較
   - `==` (等しい)
   - `!=` (等しくない)
   - `<` (より小さい)
   - `>` (より大きい)
   - `<=` (以下)
   - `>=` (以上)

2. 文字列・真偽値の比較
   - 文字列の等価性チェック
   - 真偽値の比較

## 論理演算子
1. 基本的な論理演算子
   - `&&` (AND)
   - `||` (OR)
   - `!` (NOT)

2. 論理演算子の評価順序
   - 左から右へ評価
   - ショートサーキット評価
